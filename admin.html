🔑 ورود مدیر

برای مدیریت محتوا به «توکن دسترسی گیت‌هاب» نیاز دارید. توکن فقط در مرورگر خودتان ذخیره می‌شود.

📖 چطور توکن بسازم؟ (یک‌بار برای همیشه)
  1. وارد این آدرس شوید: github.com/settings/personal-access-tokens/new
  2. در Token name بنویسید: yazd-site
  3. در Expiration مدت دلخواه (مثلاً 90 روز) را انتخاب کنید
  4. در Repository access گزینه Only select repositories و مخزن Yazd.TIMSS2027Educators را انتخاب کنید
  5. در Permissions روی Contents کلیک و Read and write را انتخاب کنید
  6. دکمه سبز Generate token را بزنید و متن توکن را کپی کنید
